Learn ConTeXt

Learn ConTeXt is an introduction to ConTeXt. ConTeXt is a document typesetting system. It generates high–quality PDF documents from source files with typographic commands.

The subtitle refers to a broader subject. The book is An Introduction to Digital Typography to the extend it explains ConTeXt. The book deals with digital typography, but this isn’t its main point. Digital typography is explained only as required by the explanation of ConTeXt.
